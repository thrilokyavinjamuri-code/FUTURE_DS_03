# HR Analytics Dashboard

## Objective
Analyze employee attrition patterns and workforce trends using Power BI to support data-driven HR decision-making.

## Tools Used
- Power BI
- HR Analytics Dataset (CSV)

## KPIs
- Total Employees
- Attrition Count
- Attrition Rate
- Average Salary

## Dashboard Features
- Attrition by Department
- Attrition by Gender
- Attrition by Age Group
- Monthly Income Analysis
- Job Satisfaction Analysis
- Interactive Filters & Slicers
- Key Insights Section

## Key Insights
- Research & Development department has the highest attrition.
- Employees aged 26–35 show the highest turnover rate.
- Lower job satisfaction increases employee attrition.
- Employees with lower monthly income are more likely to leave.
